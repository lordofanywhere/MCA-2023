Week 5: Standards in Curation

Create a metadata schema by listing the elements (or a subset of at least 5 elements)

Elements:

  • Title of the piece
  • Composer Name(s)
  • Songwriter Name(s)
  • Encoder Name(s)
  • Transcriber Name(s)
  • Release Date
  • Notated data source URI

The full schema can be explored here.
